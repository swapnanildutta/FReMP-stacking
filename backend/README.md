# Flask Backend

## Dependencies

1. Install [Python3](https://www.python.org/downloads/). Or using,

   ```bash
    $ sudo apt-get install python3
    $ sudo apt-get inRestall python3-pip
   ```

2. Install Packages.
   ```powershell
    $pip install -r requirements.txt
   ```
   or,
   ```bash
    $pip3 install -r requirements.txt
   ```
